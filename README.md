# 🏠 Mortgage Calculator

A responsive **Mortgage Calculator** built using **React + Vite**.  
This application helps users estimate their **monthly mortgage payments** and the **total repayment amount** based on loan details.

## ✨ Features

- Calculate **Monthly Mortgage Repayments**
- Show **Total Amount to be Repaid**
- Supports **Repayment** and **Interest Only** mortgage types
- Input validation for empty fields
- Responsive design for **mobile and desktop**
- Clean and simple UI

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/0xabdul/Mortgage-Calculator-Using-React-Vite.git

Go to the project folder:

cd Mortgage-Calculator-Using-React-Vite

Install dependencies:

npm install

Run the development server:

npm run dev
```

## How It Works

Users enter:

- Mortgage Amount

- Mortgage Term (years)

- Interest Rate

- Mortgage Type

The application calculates:

- Monthly repayment

- Total repayment over the loan term

- The values update dynamically based on the input fields.
